# Запуск в отдельном приложении
1. Установите необходимые библиотеки
~~~
npm install
~~~
2. Запустите сборку в режиме разработки
~~~
npm run build:dev
~~~
3. Запустите проект
~~~
npm start
~~~
3. Запустите сборку в режиме "Production" для подготовки проекта к публикации на сайте
~~~
npm run build:prod
~~~
